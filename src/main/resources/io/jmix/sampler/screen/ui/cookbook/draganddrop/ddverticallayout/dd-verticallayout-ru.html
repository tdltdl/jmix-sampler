Данный пример показывает как организовать функциональность Drag and drop, используя компонент DDVerticalLayout.

Чтобы активировать перенос компонетов из контейнера нужно установить LayoutDragMode: 

    DDVerticalLayout ddPalette = new DDVerticalLayout();
    ddPalette.setDragMode(LayoutDragMode.CLONE);

Далее установить DropHandler компоненту, который собирается обрабатывать событие падения. 

    DDVerticalLayout dashboard = new DDVerticalLayout();
    dashboard.setDropHandler(new DropHandler() {
        @Override
        public void drop(DragAndDropEvent event) {
            onDrop(event);
        }

        @Override
        public AcceptCriterion getAcceptCriterion() {
            return AcceptAll.get();
        }
    });

Методы: drop() и getCriterion() вызываются каждый раз когда переносимый компонент падает на контейнер. В методе drop() должна быть реализована логика обработки падения компонента. Метод getCriterion() позволяет фильтровать компоненты, которые падают на данный контейнер. В данном случае используется AcceptAll.get().
